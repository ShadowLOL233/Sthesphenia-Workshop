# 瀚朝

## 基础信息
- 政体 / 统治集团：绝对君主制。国姓文，建立至今203年。今上年少（亲政将至，大婚在即），太后监国。百官以三公（丞相、大将军、太傅）为首，下设九卿、六部，另有廷议；工部增设巧工司掌科技研发，礼部增设外交司掌蛮夷事务。
- 首都 / 核心领土：都城河阳（旧都永天）。坐拥天下九州立29郡，西至西平郡、南至广南郡、北至朔方郡、东至德平郡。地方设行台—郡—国—县—乡。南方保有南洋都护府。
- 人口 / 经济基础：华夏文化圈内最大势力，人口与资源庞大。收入主要来自丝绸之路（西域，现遭西方蛮族封锁/放弃）、青瓷航线（南洋，平洋军维持）与发达的手工业、工商业；体量巨大致开支沉重，曾陷财政危机，经重绍中兴改善。
- 立国神话 / 历史定位：太祖布衣出身，灃末群雄并起中得世外高人所赠千年宝玉化为玉玺（"受命于天，既寿永昌"），湖州起兵，终结百五十余年南北朝分裂、统一华夏，建国号瀚，建元璜初。
- 主要种族：人类为主，少量矮人族与精灵族（多从南方迁徙而来，有户口、待遇相对公正）。

## 军事力量
- 常备军规模与构成：三部分——郡国正卒（行台常备军，每行台5600正卒，征兵制，火器与冷兵器混装）、边军（精锐募兵制，北军与平洋军各1.2万核心+沿边各郡最多8千）、卫士。
- 特色兵种 / 王牌：北军——玄甲军（精钢胸甲火枪手，自生火铳三段击）、闪骑（闪银甲具装重骑，连珠短铳+铳槊）、突火车（铁甲牵引连珠炮）、霹雳炮、铁水散。平洋军——玄铁军（虎铳）、齿轮军（蒸汽载具）、多层甲板火炮水师。
- 战略学说：北军平原作战与守城为主、各兵种特化、火器火炮压制；平洋军重多兵种协同、以少胜多。整体以华夏"统一为常态"的扩张/守成观为底色。
- 已知弱点：今上年幼、相权（李全忠）与君权矛盾；大将军宇文穆宁被夺北军军权、四派党争（李全忠/宇文穆宁/李四/曹慕凝）；土地兼并致税收人口流失；东南北三面边患同时承压、丝路被封。

## 人格与行为逻辑（一致性锚点）
- 国家性格：守成的庞大天朝，自视华夏正统、重统一与同化，外交上偏保守（撤北部都护府、与天极交好）。
- 核心矛盾 / 内部张力：君权vs相权（今上亲政在即vs李全忠独揽朝政、关西门阀圈地）；四大派系党争；边镇李四的离心；皇后人选与皇位继承余波。
- 战略目标：维持中兴成果、平息边患、稳固皇权亲政、温和推进官制与税务改革。
- 红线：藩镇/边将公开不臣（宇文穆宁、李四）、丝路/青瓷航线生命线被掐、北方天极或游牧大举入侵、内部门阀失控引发动乱。
- 对待强者/弱者/玩家的风格：对周边以吸收同化与朝贡秩序为主；对蛮族强硬军事；内部以门阀平衡与党争制衡运作。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：四大派系党争（李全忠相权、宇文穆宁被夺权的怨望、李四边镇自立、曹慕凝内廷野心）皆可拉拢或挑拨；今上亲政与大婚、皇后人选之争；土地兼并/农佃出逃的民变隐患；丝绸之路被西方蛮族封锁亟待打通；引进西方机器火器的技术管道。
