# 故土重工

> 蜂巢联盟成员子国。由全球最大规模工业企业起家，历经「企业时期 → 军阀时期 → 国家时期（蜂巢附庸）→ 蜂巢共和国时期 → 回归故土重工并加入蜂巢联盟」。

## 基础信息
- 政体 / 统治集团：企业政府，13人董事会管理（蜂巢共和国短暂时期为独裁制）。现由故土重工、故园重工、科罗兹尼亚的保镖三大子公司主导。
- 首都 / 核心领土：一座超大型「工厂城市」（号称全世界最大规模工厂）；与戈沃非王国同岛对峙，并在各国设有分公司。
- 人口 / 经济基础：员工即居民；国内完全自循环，并持续向海外扩张子公司、哪个行业有剩余空间就做哪行，外贸活跃。
- 立国神话 / 历史定位：原科罗兹尼亚王国旗下巨型工业企业；科罗兹尼亚动乱、岛屿失管后董事会以金钱与威胁攫取「市长」之位，再以未交付军舰订单征服周边弱城；后被蜂巢帝国击败戈沃非王国后纳为自治附庸，又一度起义建「蜂巢共和国」重创蜂巢帝国，最终回归并入蜂巢联盟。
- 主要种族：待补。

## 军事力量
- 常备军规模与构成：随时期波动——军阀期2舰队/5000陆军+对峙堡垒；国家（附庸）期2舰队/1000陆军（被压缩到勉强打赢海盗）；蜂巢共和国全力期达5支舰队/10000陆军+数支海神教怪物军队（其中2舰队/3000陆军+少量怪物固守本土）。
- 特色兵种 / 王牌：「大企业家」舰队、「资本主义」舰队；蜂巢共和国期的海神教怪物军队；无可比拟的造船/军工产能（可马力全开量产战舰）。
- 战略学说：以金钱、订单与产能制胜，能用钱抹平的就不用战争；军工动员能力为最大依仗。
- 已知弱点：无军事传统、军队管理散漫，各部门军队常与执法部门起摩擦；和平期军力被宗主刻意削减。

## 人格与行为逻辑（一致性锚点）
- 国家性格：唯利是图、效率至上、以工匠精神为荣的企业型政权；用合同与金钱开疆，必要时翻脸。
- 核心矛盾 / 内部张力：宗主蜂巢帝国「咄咄逼人」的政策曾激其起义；与科罗兹尼亚王国旧账、与戈沃非王国同岛对峙的多线压力。
- 战略目标：扩大企业规模与海外市场、维护产能与自治权；曾欲彻底击败蜂巢帝国及其附庸。
- 红线：企业利益与形象（损害者由公关部「负责后事」，无论逃至何处）；自治权与产能命脉。
- 对待强者/弱者/玩家的风格：先谈钱、用订单与威胁兼并弱者；对压迫者会果断起义反击。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：世界级造船产能是各方争抢的战略资产（可下订单、可破坏）；遍布他国的分公司是渗透/情报网；与戈沃非王国的同岛对峙、与蜂巢的旧怨、海神教联系皆为可点燃的火药桶。
