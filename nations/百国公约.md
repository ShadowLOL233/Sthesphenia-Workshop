# 百国公约

## 基础信息
- 政体 / 统治集团：超国家政体 / 类联合国制度的多种族联合体。由「九之柱」为常任理事国，允许境内小国加入为邦国。核心机构：诸国联席议会（300席，统合调停、战时为最高指挥）、百国监政院（中立执法机构，可暂停任一国参会资格）。由半身人主导的塞吉亚诺发起，旧帝国（「浮晶风暴」后消亡）解体后成立，已存在近40年。
- 首都 / 核心领土：北至巴尔沃福海，南至地中海，西至卢恩萨斯岛，东至泽拉斯特亚河；资源与种族繁杂。
- **成员构成（联盟性质）**：约60+邦国，但权力集中于「九之柱」——伊斯法洛斯（精灵·实力最强）、安布兰德罗（半人马）、克罗姆达尔（矮人）、雅克联合酋长国（鱼人）、阿克里沙司（羽翼人·物流商业）、泽瓦特评议国（半兽人·基建常备军）、德鲁里科斯克（蜥蜴人·宗教王国）、杰尔查/扎基尔（巨人）、塞吉亚诺联合王国（半身人+人类·发起国）。九之柱瓜分约240席，小邦另组「自由联盟」抗衡。
- 人口 / 经济基础：分工制——伊斯法洛斯/克罗姆达尔/塞吉亚诺担制造业与工业，安布兰德罗/阿克里沙司/雅克担陆/空/海商贸，泽瓦特/德鲁里科斯克/扎基尔担基建与常备军。
- 立国神话 / 历史定位：古时「百国之地」小邦曾被旧帝国征服为「百国郡」；旧帝国于「浮晶风暴」灾难中彻底消失，百国从废墟重生，塞吉亚诺半身人国王埃伽卢姆·墨洛温自称护诸国主、颁《百国公约宣言》立国。
- 主要种族：精灵、半人马、矮人、鱼人、羽翼人、半兽人、蜥蜴人、巨人、半身人、人类（九之柱各族）。

## 军事力量
- 常备军规模与构成：无统一军队，战时由联席议会指挥「公约联合军」，各柱出兵。公约防备军主力为安布兰德罗半人马30万骑兵；常备军由泽瓦特、德鲁里科斯克、扎基尔分担。
- 特色兵种 / 王牌：精灵闪银甲战士+魔法火器+大型风帆战舰（数量稀少）；矮人群峰氏族兵+铁甲舰队+最先进飞艇舰队（雷击/旋翼/迅雷）；鱼人水军（鱼骑手、巨兽战争机器、海洋祭司召台风、利维坦）；半人马枪斧骑兵；半兽人近战陆军+试验性陆行舰雏形；蜥蜴人神祝战士。
- 战略学说：分工互补、各族按特长（精灵海运护航、矮人铁甲与制空、鱼人制海、半人马陆地机动、半兽人攻坚基建）。
- 已知弱点：缺统一指挥、九之柱内讧（尤其对伊斯法洛斯傲慢的不满催生自由联盟）；多国军种残缺（半兽人无海空、蜥蜴人/半人马仅陆军、塞吉亚诺几无军队）；精灵闪银/魔法火器技术失传、产量极低。

## 人格与行为逻辑（一致性锚点）
- 国家性格：松散多元的多种族议会联合体，半身人式调停文化为底色；理性时务实、博弈时各怀算盘。
- 核心矛盾 / 内部张力：九之柱 vs 自由联盟（小邦要求选举/任期、断九之柱合法性）；精灵伊斯法洛斯的傲慢自私招致众怨；各成员国内部派系（蜥蜴人君权神权、半兽人集权/保守/改革、羽翼人女王改革、塞吉亚诺半身人与人类融合之争）。
- 战略目标：维系联盟运转、由监政院调停冲突、避免九之柱矛盾激化为战争；推动基建与工业体系化。
- 红线：《百国公约宣言》（保障每邦国最少一席）与监政院执法权；九之柱守护自身既得席位与权力。
- 对待强者/弱者/玩家的风格：内部以席位实力定话语权、强者瓜分权力；监政院标榜绝对中立；对外整体取决于哪几柱主导，商贸大国倾向往来。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：扶植自由联盟撼动九之柱、或挑动小邦反伊斯法洛斯；各柱内部派系（蜥蜴人内战边缘、半兽人/半身人改革之争）可介入；矮人先进飞艇/铁甲与卢恩工艺、精灵失传的闪银魔法火器、鱼人深海之母/海洋祭司/利维坦之谜皆为利益与冒险钩子；监政院可被游说取消某国资格；联盟松散指挥利于分化各个击破。
