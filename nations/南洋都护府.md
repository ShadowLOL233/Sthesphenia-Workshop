# 南洋都护府

## 基础信息
- 政体 / 统治集团：都护府（瀚朝直属军政机构）。南洋节度使（兼南洋都护、宁涛伯）李四一人掌握辖区内一切军政大权，只对皇帝本人负责。
- 首都 / 核心领土：旧海龟王国故地，全府划分十一州（含咸昌、广南、朔同、坪穗等边郡，邢州原名哈塔木耳），临龙兴湾与金玉洋。
- 人口 / 经济基础：以人类为主，少量鱼人与半兽人。沿袭瀚朝小农经济——农业、沿海捕鱼业、手工业为底；最大收入来自青瓷航线（由南洋水师维护，含朝廷配额输入与水师抽成）。
- 立国神话 / 历史定位：前身为海龟王国；重光六年中宗派天军南下剿灭海龟国后设府。修明年间南洋大叛乱，前节度使郭伦贤被围自刎；平南将军李四率四郡边军平叛，修明十一年灭海龟，受封新任节度使。
- 主要种族：人类（绝对多数），少量鱼人、半兽人。

## 军事力量
- 常备军规模与构成：平洋军（现称南洋军），瀚朝边军之一，核心约一万两千精锐，可加配沿边郡正卒与边军组成混合部队；含基础步兵玄铁军、齿轮军（蒸汽载具）、装备数十门火炮的多层甲板水师。
- 特色兵种 / 王牌：玄铁军（持短管火器"虎铳"，发射火药弩箭与梨花弹，远破铁甲近破军阵，靠战术与配合打高战损比）；齿轮军（圆盘齿轮机车，覆木甲、环布六门连珠铳360度无死角）；新锐多层战船水师。
- 战略学说：重各兵种协同配合（区别于北军的单兵种特化），依托地形与火炮优势以少胜多（嘉宁四年烁顶山役三千破一万五千南蛮联军）。
- 已知弱点：飞艇与多数齿轮设备仍处实验阶段；内部海神教渗透、原住民排外；节度使产业脱离兵部户部管控，军纪松弛（谎报战功、克扣货品、私售军资）。

## 人格与行为逻辑（一致性锚点）
- 国家性格：守成+半自立的边镇。表面忠于瀚朝，实则山高皇帝远、自成体系。
- 核心矛盾 / 内部张力：瀚派（执政，李四+南洋军）vs 海龟派（在野旧贵名士，今与瀚派合作）vs 海神教派（隐秘渗透，高层10%基层50%，密谋颠覆建国）；李四重兵在手对朝廷构成"心腹之患"。
- 战略目标：抵御南部海盗与蛮族、维持青瓷航线收益、安定刚平叛之地；李四个人则在稳固自身权位与自主性。
- 红线：海神教/海龟残党再叛、青瓷航线被切断、朝廷强行收夺节度使军政权。
- 对待强者/弱者/玩家的风格：对朝廷阳奉阴违、以恭顺换自主；对蛮族海盗强硬军事；对地方名士怀柔招抚（一石二鸟）。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：海神教密谋颠覆（教主行踪不定、掌禁术）可被玩家引爆或揭发；李四与朝廷的离心倾向可被挑拨或拉拢；青瓷航线收益与海盗问题；飞艇技术"来自海上遥远之地"的引进线索；原住民排外情绪可被别有用心者利用。
