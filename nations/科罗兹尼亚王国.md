# 科罗兹尼亚王国

## 基础信息
- 政体 / 统治集团：三元君主立宪制（独创，介于专制帝国与民主共和之间）。王国元首"萨尔森"（现任洛法塔姆二世，鲁登瑞思堡家族）、王国议会（各民族邦票选代表，选首相、制定法案）、皇家法院与皇家民族理事部三方制衡。现任首相亚历山大·武库莱波夫，民族事务管理官特里克斯·俄列戈，法院院长奥普列克·冯·鲁登瑞思堡。
- 首都 / 核心领土：拉尔特民族邦鲁登瑞思堡王国区。行政由"民族邦+王国区+自治领"并行：7个民族邦（拉尔特/多卡列姆/布卡里塞/布登/萨菲托利安/利萨/比代尔尼亚）、3个王国区（鲁登瑞思堡/君主港/尼克德）、3个自治领（波里斯/克里克莱人/喀什尼尔自治斯坦），共13个行政单位。
- 人口 / 经济基础：渔业与矿藏丰富，浮空渔业近年发展迅速。多卡列姆为粮仓与畜牧中心；布卡里塞为造船/远洋渔业/资本主义萌芽地；布登为空渔业中心；利萨手工业、比代尔尼亚商业。经济仍含落后农奴制（多卡列姆为"重灾区"），正推行废除农奴制改革；资本主义已生根（皇家银行、实业家涌入），但也出现残酷"奴隶制"种植园。
- 立国神话 / 历史定位：四百年历史、四代王朝，位于欧西尼亚极西之地，西接巴尔沃福海、东连沃查理特山脉，北接百国郡（百国公约）。原为康拉德帝国"极西郡"，对旧帝国的缓冲带。康拉德崩溃后卡姆斯尔大公与皇室观察官兰克考尔宣布独立建"科罗兹尼亚（边缘之地）王国"；末代公主联姻使帝国正统流入。历经普列克→卡兰多姆→（大分裂期铎莱特）→鲁登瑞思堡王朝；455年"民族大改革"创立民族邦制度与三元君主立宪制。
- 主要种族：六大主要民族——布卡里塞（航海/远洋渔，三月教，开放）、多卡列姆（畜牧/农奴主源，二月教，保守顺从、阶级森严）、拉尔特（王室民族，手工业/小农，虔诚三月教/天文地理）、喀什尼尔（沙漠陆行鸟骑手，穆马教，桀骜排外，含极端派回流隐患）、布登（高地空渔，空神教，自由）、萨菲托利安（狩猎/精密制造，重效率轻传统）。非人族：克里克莱人（亚熊人，天生战士、自治领、宗族文化）、猪人（流亡帮派、被"城镇猎人"猎杀）、血族（图里曼夫血皇国南逃贵族，巨富、长寿、强体力、惧阳光嗜血月）。

## 军事力量
- 常备军规模与构成：陆军为最古老军种，从康拉德古典方阵演进到线列步兵+散兵战。核心为"科罗兹尼亚国防军"——七万精锐职业军，皇家军部统一配发装备，靠严明纪律与密集齐射作战（第一次喀什尼尔战争后设立，对标大汗亲军）。另有大量装备落后、士气低的农奴兵（随改革递减）。海军因海岸线短而弱小、走精简路线（超一级与二三级风帆战舰），以防卫作战为主。
- 特色兵种 / 王牌：萨尔森卫队（元首嫡系精锐、薪三倍、突击力量）；普里季克兵团（"黑手套"，亡命徒雇佣军团、灰军服+短斧+滑膛枪、高死亡率高先进装备率、拥空艇战舰使用权、名义不属王国但军部可调）；亚人军（猪人任线列/散兵、亚熊人任大型生物骑兵/近战、血族任渗透刺杀间谍，设调节机构防内斗）；萨菲托利安人组成的精锐散兵。
- 战略学说：海军以防卫而非争夺制海权为主。陆军线列齐射+散兵。皇家军部统合三军并协调各民族邦邦军，但实际前线指挥权在各民族邦邦军指挥机构；对外战争民族邦可拒派军，本土防御则不得拒调。设"民族邦驻防军"由中央应邦请求弹压叛乱邦军。
- 已知弱点：民族邦邦军几乎独立于中央、历史上屡有军官拥兵自重；海军弱小、海岸线短；部分民族邦军仍用火绳枪；农奴兵拖累战力；喀什尼尔自治斯坦宗教激进派分裂危机；士绅守旧派可能放手一搏发动内战。

## 人格与行为逻辑（一致性锚点）
- 国家性格：求变图存的改革型君主立宪国。萨尔森洛法塔姆二世以"随时代变革"为旗、忧惧"百年前灾祸重演"、以理性与造福苍生自勉；整体务实、重经济发展、尊重多元民族信仰。
- 核心矛盾 / 内部张力：废除农奴制改革 vs 士绅同盟/守旧派（已策划刺杀、阻挠民事官入境、阴谋内战）；中央集权 vs 民族邦高度自治/邦军拥兵自重；喀什尼尔传统宗教激进派 vs 世俗化改革；资本主义发展 vs 新生奴隶制种植园；多民族认同（民族邦团结党理想）与现实民族差异的鸿沟。
- 战略目标：完成农奴制改革与生产方式转型、推进资本主义与宗教世俗化、巩固"科罗兹尼亚人"民族认同与中央对民族邦的调控，避免落后于世界而被外敌"欺凌屠戮"。
- 红线：威胁王国整体安全的本土入侵（民族邦无权拒调）；民族邦邦军反叛/拥兵自重（出动驻防军武装缴械）；守旧派发动内战。
- 对待强者/弱者/玩家的风格：对强邻莫林多弗——可结盟对象（被其拉拢，但自身空军反令对方忌惮）；对百国公约接壤；对内部改革阻力恩威并施（赞赏交地契的开明地主、推广典型，弹压刺杀阴谋）；对非人族态度因邦而异（克里克莱人获自治领与平等权，猪人遭猎杀，血族为帮派金主）。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：农奴制改革的全国撕裂（守旧派士绅、波里斯自治领拒入境、喀什尼尔刺杀案——可被外力激化为内战）；民族邦邦军独立性——可扶持某邦拥兵自重或分裂（喀什尼尔/波里斯）；喀什尼尔穆马教极端派回流的宗教火药桶；血族金主与猪人帮派的城市暗线；浮空渔业/浮晶民用化的技术合作钩子；莫林多弗-科罗兹尼亚-百国公约三角外交是大陆西部核心杠杆。
