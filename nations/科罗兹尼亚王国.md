# 科罗兹尼亚王国

> 已据 GM 口述要点更新为"当前形态（公历 1890）"。旧 docx 的详尽底盘（三元君主立宪+民族邦制、各民族/军事/派系）仍有效；本档案记录其**改革完成 → 第一次科莫战争 → 与莫林多弗联姻缓和**的当前态。

## 基础信息
- 政体 / 统治集团：三元君主立宪制（独创，介于专制帝国与民主共和之间）。元首"萨尔森"现任 **洛法塔姆三世**（鲁登瑞思堡家族；前为洛法塔姆二世）。王国议会（各民族邦票选）+ 皇家法院 + 皇家民族理事部三方制衡。
- 行政区划：7 民族邦 + 3 王国区 + 自治领（改革后**喀什尼尔由分裂危机转为新设自治区**而缓解）。
- 首都 / 核心领土：拉尔特民族邦鲁登瑞思堡王国区。西接巴尔沃福海、东连沃查理特山脉、**北接莫林多弗**、**西南方为百国公约**、南邻新泽威尔共和国。
- 人口 / 经济基础：渔业与矿藏丰富、浮空渔业发展迅速；**改革后迈入新兴浮晶工业**（不及有冬之桥加持的莫林多弗，但已是主要新兴浮晶工业势力之一）；资本主义生根。**农奴制改革已完成**（曾"重灾区"多卡列姆等地阵痛已过）。
- 立国神话 / 历史定位：四百年四代王朝，欧西尼亚极西之地；原旧帝国"极西郡"缓冲带，与**莫林多弗同为旧潘泽尔帝国的继承王朝**——两国皆自称旧帝国正统，此为科莫世仇之根。
- 主要种族：六大民族（布卡里塞/多卡列姆/拉尔特/喀什尼尔/布登/萨菲托利安）+ 非人族（克里克莱亚熊人有自治领、猪人、南逃血族贵族）。详见旧 docx。

## 军事力量
- 国防军（七万精锐职业线列军，皇家军部统配）、萨尔森卫队（元首嫡系）、普里季克兵团（"黑手套"亡命雇佣军团，拥空艇战舰）、亚人军（猪人线列/亚熊人生物骑兵/血族渗透）、萨菲托利安精锐散兵；海军走精简防卫路线（海岸线短）。
- 战略学说：陆军线列齐射+散兵；皇家军部统合三军并协调各民族邦邦军（前线指挥权多在邦军）。
- 北方边境军镇见下文"开局态势"。
- 已知弱点：民族邦邦军半独立、历史上拥兵自重；海军弱小、海岸线短；与莫林多弗的长期边境消耗。

## 人格与行为逻辑（一致性锚点）
- 国家性格：求变图存、改革有成的君主立宪国；洛法塔姆三世务实、重经济发展与休养生息、忧惧落后被屠。
- 核心矛盾（1890）：**与莫林多弗争旧帝国正统的世仇**（虽联姻暂缓，根因未除）；民族邦自治 vs 中央调控；改革后社会的再整合。
- 战略目标：内战后休养生息、巩固改革与新兴浮晶工业、维持与莫林多弗的缓和、稳固"科罗兹尼亚人"民族认同。
- 红线：莫林多弗（或其南方派系）重启全面战争；民族邦反叛；改革成果被颠覆。
- 对待各方：对莫林多弗——世仇但已联姻缓和、乐见继续好转；对新泽威尔——友好（贸易+情报）；对百国公约——尚无有效外交。

## 沿革与开局态势（公历 1890）

### 改革的结局
农奴制改革**经历局部内战**而完成：**士绅守旧派遭清算**；喀什尼尔分裂危机由改革后政府**设自治区**而缓解。改革后科罗兹尼亚国力**超越内战前**。

### 科莫世仇与第一次科莫战争
- **根源**：科罗兹尼亚与莫林多弗**同为旧潘泽尔帝国的继承王朝，皆自称正统**——边境摩擦自潘泽尔帝国解体起从未停歇，但国家级冲突长期因双方各自内乱而压着没爆。
- **临界点**：莫林多弗内战（南北战争）结束 + 科罗兹尼亚改革成功 → **两个统一且蒸蒸日上的旧帝国继承王朝再无法忽视彼此** → **第一次科莫战争**爆发。
- **细化的导火索（GM 拟，待你确认）**：真正点火的不是布伦特里亚政府，而是**莫林多弗内部的南方王国派/独立派**。这些南北战争的败方残党，与战后**流亡科罗兹尼亚"北方边境领"的反莫南方贵族本是同一阵营**（隔萨克森边境一分为二）。他们沿萨克森边境制造、放大流血事件（边民袭扰、争议边镇、扣留浮晶商队、围绕"收容流亡贵族"的口实），并把事态包装成**关乎旧帝国正统颜面、莫中央退无可退**之局——逼布伦特里亚政府不得不应战。其算计是**以战养战、放血北方皇权派的工业产能与人力**（南方不缺人口，北方又不能在国战中束手不管）。科罗兹尼亚则以自身"旧帝国正统"之名应战。→ **南方王国派/独立派的暗中助力，是第一次科莫战争的重要导火索。**
- **结果**：战争**无明确胜负**；唯一产物是双方在各自国境线上设立两个**缓冲附属国**——**上萨克森自由邦**与**下萨克森自由邦**。

### 北方边境领
科罗兹尼亚"北方边境领" = 面向莫林多弗前线的**边境邦/军镇**（含下萨克森自由邦），**早在第一次科莫战争前即已设立**（备战之用）。如今小规模边境冲突屡见不鲜——**多由莫林多弗南方王国派/独立派主使，而非莫中央**。

### 联姻缓和（开局现状）
- 沿上下萨克森自由邦的大小边境战争已**僵持十余年**。
- 莫林多弗正全力应付东方的**巴塞尼亚帝国**；洛法塔姆三世不愿让世仇无休止延续。
- 故科罗兹尼亚在莫备战巴塞尼亚期间**提出联婚**，获莫接受——**科罗兹尼亚王子 × 莫林多弗公主**联姻，敌意与冲突**暂时缓和**。根因未除，但内战后需休养的科、备战巴塞尼亚的莫**都乐见此转变**，可见未来内继续好转。

### 周边关系
- **新泽威尔共和国**（南邻、新兴势力）：**良好关系**，活跃的贸易 + 情报交易。
- **百国公约**（西南方，非北邻）：彼时百国公约尚未成整体、深陷内乱 → 科罗兹尼亚**未能与之建立有效外交**。
- **戈沃非王国 / 故土重工**"原属科罗兹尼亚"系**古早地图的废弃设定**，玩家未据新图改动 → **不采纳**。
