# 千月暮雪（暮雪国）

## 基础信息
- 政体 / 统治集团：邦联议会制。狐族「暮雪幕府」为邦联级共同执行机构，盟主／将军暮雪如月坐镇首都大京；幕府行寡头奉行制（五奉行：社/天领/勘定/观星/水溪），天皇为傀儡。下辖七个超自然族群成员国。已向天极帝国称臣朝贡。
- 首都 / 核心领土：首都大京；狐族本土为岚华圣国（旧暮雪国核心，信仰心脏神田稻荷大社）。千坂群岛含北畠、早形、夜川、与那国等群岛，温带雨林、可耕地较多但极缺金属矿产、长期靠海外输入。
- 人口 / 经济基础：约30万神社／神明规模的神道社会。经济靠税收、神社奉纳、外来旅游消费、远洋渔业与手工业，及多条对外贸易航线（菲尔佐甘大陆、普拉梅斯半岛、北海、天极帝国直贸）。
- 立国神话 / 历史定位：扶桑国解体后内战，神田稻荷大社狐狸凭依被杀巫女之身、借稻荷神之力化为人形巫女，沿神社扩张、臣服天极换得战舰，数十年内战于冬末结束统一（故名「暮雪」），后邦联化让名于全联盟。
- 主要种族：狐族（原住民、半兽人，1-7尾，雌雄皆显女性特征）、鬼族（角、力大、永不说谎）、妖怪（弱小神明到灵体总称、月下更强）、乌鸦天狗（羽翼人、翼展13m、星象气象专家）；另含灵体妖怪、亡魂、梦幻妖怪等成员国族群。

## 军事力量
- 常备军规模与构成：幕府军（陆军，弩箭长弓+铁炮队+神道祝福甲）、幕府舰队（大量五/六级风帆舰，旗舰为购自天极的三级战列舰「长风丸」，分四支舰队+商船护航队）、暮雪卫队（神道巫女精锐，约3000人，「狐火」）、幕府天军（试验性弩炮飞艇）、神道军（巫女宗教部队，专猎外来传教士）。
- 特色兵种 / 王牌：暮雪卫队「狐火」——护盾可挡火枪、吸生命充能、弓镰可斩金属、林间如鬼魅；铁炮队；各成员国特色（鬼族重装铁拳、妖怪术法弹幕夜战、天狗制空侦察、灵体/亡魂构造体之军、梦幻幻术乱敌）。
- 战略学说：本土依托城墙天守防御、海岸警备微型巡逻舰+远洋舰队护贸；邦联诸族按特长分担制空/夜战/重装/构造体/惑敌。
- 已知弱点：缺金属矿产难量产；浮晶/飞艇技术原始；成员国松散、狐鬼内讧；天极属国身份受制；魔力迸发地带魔兽永久内患牵制神力。

## 人格与行为逻辑（一致性锚点）
- 国家性格：狐族狡黠机敏、长于权谋虚饰，老练却善变；整体清雅而神秘，被同一片迸发之海孕育又围攻的存续共同体。
- 核心矛盾 / 内部张力：狐鬼世仇（机变虚饰 vs 永不说谎）；守旧派（幕府士族保权、暗杀革新派）vs 革新派（资产阶级改革）vs 激进派（亲天极、欲并入天极）；邦联成员国各自为政。
- 战略目标：维系邦联存续、抵御魔兽内患；扩展海贸（含筹备东南方向新航线开拓舰队）；在天极朝贡体系内自保发展。
- 红线：稻荷神/神道信仰与暮雪如月血统正当性；神道军清剿一切外来传教士。
- 对待强者/弱者/玩家的风格：对天极恭顺称臣；对外客以贸易/旅游接纳、善以承诺与交易玩弄混乱；对传教渗透零容忍。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：狐鬼世仇可激化或调停；守旧/革新/激进三派权斗（尤其激进派的天极并入诉求）；缺金属矿可作贸易筹码；东南新航线开拓可同行或截胡；魔力迸发地带的魔兽/神祇/构造体之谜；触碰神道禁忌将招神道军猎杀。
