# 东齐（曹氏）

> **骨架档案**：据 `setting/瀚末编年.md` 的**建国期**记述搭建。瀚朝解体后天下四分中的**东陲外戚政权**。现任君主、技术现状、当代国力等公历 1890 年的演化**待 GM 补全**。

## 基础信息
- 政体 / 统治集团：外戚**曹氏**所建。**挟瀚室文氏幼主以奉正朔**（幼主为傀儡，政柄尽出曹氏）。奠基者**曹慕凝**——以椒房之亲入枢，早断瀚必亡，不救瀚、不附李，阴蓄其力以俟天下之变；其人于朔东讨之役中病殁。族大而杂、派系丛生，**全赖主君一人之威维系**。
- 首都 / 核心领土：帝国**东陲**，多山（层峦叠嶂）、大河贯其中、关隘扼要；凭山为城、据河为堑。地图疑为"东"（首府**关阳**，辖武府/宁州/邳州/福桐县）。
- 人口 / 经济基础：东撤时**挈府库图籍、巧工良匠、积年财货以去**，故立国未久即仓廪已实、器械已备，非草创之比。
- 立国神话 / 历史定位：河阳之变中曹李火并不胜，慕凝从容引众东走、挟幼主据东陲，立国曰东齐。既据东方第一等之强、又奉瀚之正朔，**隐然以"天下共主"自居**，而其实偏处东隅、志在自保。
- 主要种族：华夏人。

## 军事力量
- 常备军规模与构成：以**东北镇兵**为爪牙——瀚之劲旅，世戍东陲御东北之患，甲坚卒悍，不在北军之下；以**禁军余烬**为腹心。整体**守险有余，出险则患其老迈**。
- 特色兵种 / 王牌：东陲山河险隘的防御体系（凭山为城、据河为堑，要害不可猝拔，连朔太祖累月攻之不能克）。
- 战略学说：奉正朔以收人望、据险以自保；非进取扩张，志在保全曹族于东陲。
- 已知弱点：**偏处东隅、志在自保**，无并吞之势；曹族族大而杂，**赖主君一人之威**——主心去则诸将离心（可破之隙，然其险终不可猝拔）；镇兵老迈、出险作战乏力。

## 人格与行为逻辑（一致性锚点）
- 国家性格：挟正朔而自居"天下共主"、实则偏安自保的**东方门阀—外戚政权**；深沉、务实、长于谋身渡险。
- 核心矛盾 / 内部张力：曹氏诸派系内斗（主威一去即裂）；**所挟文氏幼主的下落与正朔名分**（"正朔之争"伏线）；据险自保之惰性 vs 名义共主之野心。
- 战略目标：保全曹族与东陲之安；以"奉正朔"对抗朔之"再造华夏"，维持名分上的天下共主地位。
- 红线：东陲险要被攻入；文氏幼主名分被夺或被外人利用（动摇其正朔合法性）；曹族内部派系失控。
- 对待强者/弱者/玩家的风格：对朔——划险相持、以正朔相抗而避野战；对凉/瀛——以名分周旋；对内诸派——以主威与利益平衡维系；总体守势、谋身为先。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：**曹族派系内裂**（主威一去即可被外力撬动）；**所挟文氏幼主**——正朔合法性的活筹码，可被各方争夺/扶立/揭破，是"正朔之争"的核心伏线；东陲险固=易守难攻的战略缓冲；"奉正朔 vs 朔之再造华夏"的名分战可被玩家介入。
