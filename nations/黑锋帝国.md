# 黑锋帝国

## 基础信息
- 政体 / 统治集团：非世袭君主专制；皇位选举制（笔试+体力+对改造生物掌控度），皇子需经平民抚养与"黑谷七日"考验才有继承权，外人理论可参选（260年来无外人当选）；承相主管农/军/技三司，监查司反腐、通报司通报民意。
- 首都 / 核心领土：黑锋峰一带（最高学府席兹大学院所在）；南方为最大农业产地与商业港口（断罪教主基地）。
- 人口 / 经济基础：以阿佐兰尼人及其改造体为主；第一产业（农业）发达、轻工业薄弱但自给；政府定期屯粮与购精神药物防"小意外"；主要出口金属矿石及工艺品、寒带作物/草药/香料、寒带生物肉品；按生产要素分配（多劳多得）。
- 立国神话 / 历史定位：旧帝国科考队覆灭后，哈桑部落回收矿井残存科技起家，约1580年强盛、1660年统一部落建黑锋帝国；1730年雷坎尔将军与五代女皇通过解析深渊之井旧帝国科技带来生物科技革命，1795年七代皇帝登基。
- 主要种族：潘斯派特（人类原住民）、潘斯派多森（杂种人/改造人）、文多森（公民权怪物）、多文（非公民权怪物）。

## 军事力量
- 常备军规模与构成：13个军团，皇家议事会与将军最高决策。第一军团"重锤"（人数最多、兵种最全，含少有的空中力量与海军卫队，主打人海消耗与协同）；第二军团"红天使"（400人精锐，护卫王室，精制金属装备）；第四军团"炎息"（死刑犯/违法者组成，与断罪教关系密切）；第六军团"铁幕"（镇暴驻军）；第七军团"渗透者"（全员潘斯派多森的变容间谍）；第九军团"惩戒部队"（督战队）；第十一军团"剑鱼"（海军主力）。
- 特色兵种 / 王牌：生物科技兵器——黑水（集群寄生虫，母核难毁、耐高低温、强酸克制）、深渊生物、深渊骑士（黑水寄生的皇室成员，力量/寿命远超深渊生物）；精良锻造装备。
- 战略学说：军团协同+生物科技不对称战力；重锤以人力优势协同其他专精军团取胜。
- 已知弱点：轻工业薄弱、依赖粮食贸易与屯粮；断罪教坐大于南方港口（一旦造出"神明"或圣战将威胁全南大陆）；炎息军团/断罪教与惩戒部队的内部张力；黑水失控的污染风险。

## 人格与行为逻辑（一致性锚点）
- 国家性格：尚武、利他主义集体文化、技术（生物科技+锻造）自豪、专制而精英筛选。
- 核心矛盾 / 内部张力：断罪教（信仰多门、炎息军团多为信徒）日益坐大 vs 王室与惩戒部队的监督；生物科技伦理与黑水失控风险；选举君主制的精英筛选 vs 皇族血脉。
- 战略目标：维系帝国统一与生物科技/锻造优势、保障粮食安全（屯粮+贸易）、防范断罪教威胁、向海外溯断罪教之源。
- 红线：王室安全受威胁（红天使死保）、黑水/深渊技术被外部夺取或失控、断罪教完成"神明"发动圣战、粮食命脉被断。
- 对待强者/弱者/玩家的风格：对邻国以粮食贸易与锻造/科技交流为主，务实；对强者愿技术交流，对威胁王室或技术者以军团与生物兵器强硬回应。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：黑水/深渊骑士生物科技是危险的力量来源与道德冲突点；断罪教"造神"圣战阴谋是可引爆的大威胁线（玩家可揭露、阻止或助推）；精良锻造工艺/寒带特产为贸易钩子；渗透者间谍线可被卷入；激怒点为夺取生物科技、威胁王室或助断罪教。
