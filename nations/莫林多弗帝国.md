# 莫林多弗帝国

> 已据 GM 口述要点更新为"当前形态（公历 1890）"。前期沿革（潘泽尔帝国→邦联→四十年战争→第二帝国）见旧 docx；本档案记录**第二帝国后续：南北战争 → 布伦特里亚一世重整 → 大规模工业化/浮晶化 → 灭巴塞尼亚**的崛起轨迹。

## 基础信息
- 政体 / 统治集团：**君主主导的半宪政政府**。梅斯特里亚三世改革整合了**北方皇权派 + 东方共和派**；其后由 **布伦特里亚一世** 带领走出内战、强化国力。**布伦特里亚 = 梅斯特里亚家族与北方皇权派显赫家族联姻而生的新 house**（象征皇室与北方改革势力的结合）。保留帝国议会两院 + 选帝侯高台 + 枢密院等架构（改革后皇权主导）。
- 首都 / 核心领土：帝都奥斯兰德；古核心施陶芬尼亚。环奥肖罗山脉、三面环海，南接科罗兹尼亚。**威杨共和国已于梅三改革中回归、并入帝国**。
- 人口 / 经济基础：**19 世纪中叶后迈入大规模工业化与浮晶化**（冬之桥协议的红利）；吃到大量农耕人口红利；矿产丰富（北境维特霍芬家族最大浮晶矿区）。
- 立国神话 / 历史定位：源自旧帝国阵营的古老国家，历潘泽尔帝国灭亡→邦联→四十年战争而重建为第二帝国，"恢复往日荣光"。现已是**统一且崛起中的北方工业强权**。
- 主要种族：人类主体；**亚人战争已结束**（布伦特里亚一世期间基本镇压）——帝国以"拉拢愿加入帝国的部分半兽人/半精灵"**分化亚人**而平定；今帝国政府**已抛弃人类至上主义**，**合格亚人（符合政府需求者）可进入人类社会任职**；仅余零星抵抗，至多微弱影响地方治安。

## 军事力量
- **莫林多弗铁军**：第一世代机械化部队总称——陆行舰、浮晶飞艇、各式装甲载具、军用火车。是灭巴塞尼亚、对抗科罗兹尼亚的主力。
- 空军底蕴：四十年战争锤炼的浮晶飞艇传统（云海舰队等，详见旧 docx；改革后指挥权归属待更新）。
- 已知弱点：内战耗损（尤其皇权派资源人力）；旧工业贵族 vs 新资本的军工/政治路线之争；南方残余反莫势力外流至科罗兹尼亚。

## 人格与行为逻辑（一致性锚点）
- 国家性格：崛起的工业帝国，自视旧帝国正统继承、向东扩张、以工业碾压旧时代魔法范式。
- 核心矛盾（1890）：**旧北方工业化贵族 ↔ 冬之桥扶持的新一代工业家/资本家**（战后不可调和）；内战后南方整合的余波。（亚人战争已平、退为次要。）
- 战略目标：巩固统一与工业化成果、制衡南部改革有成的科罗兹尼亚、消化向东扩张（巴塞尼亚故地）的战果。
- 红线：科罗兹尼亚支持的反莫南方贵族复燃内乱；工业贵族/新资本之争失控；外力干涉其东扩。
- 对待各方：视科罗兹尼亚为头号战略对手；倚重冬之桥的技术；对百国公约相对轻视。

## 开局态势（公历 1890）
- **南北战争**（梅三在位时爆发的帝国内战）：皇权派+共和派胜、南方王国派/独立派遭重创但近乎耗尽皇权派资源；**部分南方贵族流亡科罗兹尼亚，成为科"北方边境领"铁杆反莫力量**。
- **世仇/暂缓 = 科罗兹尼亚**：科莫同为旧潘泽尔帝国继承王朝、皆争正统，故为世仇；两国统一崛起后爆发**第一次科莫战争**（无明确胜负，沿边境立**上/下萨克森自由邦**两缓冲附属国）。**关键：边境战争多由莫内部的南方王国派/独立派暗中挑动**（与流亡科罗兹尼亚的反莫南方贵族同源），意在"以战养战"放血北方皇权派工业产能。沿萨克森僵持十余年后，因莫备战巴塞尼亚，**科主动提出联姻（科王子×莫公主）获莫接受，敌意暂缓**、可见未来续好转（根因未除）。
- **已灭巴塞尼亚帝国**：莫向东扩张，以铁军碾压"魔法×火器"的巴塞尼亚（诺尔瓦尔帝国之后身、曾兼并图尔坎汗国）；冬之桥学会为技术后台。此战震动全瑟斯芬尼亚（见 timeline / 冬之桥学会档案）。
- **冬之桥协议**：布伦特里亚一世期间，帝国允许冬之桥在境内设有限科研设施+封地，换取技术；催生学会主导的改革派+工业资本派壮大。
- 其余：**百国公约**威胁相对次要；**不列颠尼亚王国**隔海相望。

## 已确认（2026-06-25）
- **类型/状态**：原玩家**已搁置** → **GM 接管为 NPC**（玩家回归则交还）。
- **布伦特里亚一世**：梅斯特里亚家族 × 北方皇权派显赫家族**联姻而生的新 house**（皇室与北方改革势力的结合）。
- **科罗兹尼亚**仍为**王国**（前述"帝国"系笔误）。

- **亚人战争（已确认结束 2026-06-25）**：布伦特里亚一世期间以"分化+镇压"基本平定（拉拢部分半兽人/半精灵）；帝国已弃人类至上主义，合格亚人可入人类社会任职；仅余零星、微弱影响治安的抵抗。**叙事钩子**：被分化拉拢 vs 残余抵抗的亚人、血族的统治欲，仍是可被外力撬动的暗线。
