# 极地工程协会

## 基础信息
- 政体 / 统治集团：寡头（技工）专制。最高领导者「大技师」艾希·乌尔苏斯（Icey Ursus）拥有最高指挥权与直属武装督察；下设四位总顾问（工程／民事／军事／外交）组成顾问团，工程师议会已名存实亡。
- 首都 / 核心领土：拥有一座城市与两座小镇，全部依托「能量塔」建造；首都坐落于一处大型浮鲸迁徙点。前身为旧帝国北极开拓队，现处「大北进时期」向北极继续扩张。
- 人口 / 经济基础：主体为标准智人（旧帝国移民与后裔）。经济基础为定期捕猎浮鲸（生物香料「浮香」、鲸油）与地下浮晶矿业，并接受器械生产加工订单；完全配给制度。
- 立国神话 / 历史定位：旧帝国崩溃后被困北方的猎人与一批被迫害的异见学者结合而成，经猎点→崩溃→议会→专制过渡→专制→大北进诸时期。形成「为科学牺牲是光荣而理所当然」的争议文化。
- 主要种族：标准智人为主；被征服的三支土著为底层劳力——极地贝尔族（北极熊系半兽人，约240cm/350+kg）、极地枭族（北极枭系羽翼人，娇小善突袭）、极地沃尔夫族（北极狼系半兽人，迅捷暴躁）。

## 军事力量
- 常备军规模与构成：实质唯一军队为「雪拓探险队」，由保罗·贝尔（Paul Bel，前帝国军人、总先锋）掌握；内阁派近期另建「武装技师」但尚未成战力；大技师另有直属「武装督察」。
- 特色兵种 / 王牌：北极狼陆战猎手（雪橇散兵，线膛燧发枪+枪弩+掷弹发射筒）、北极枭侦查猎手（连发火枪、敌后精确打击）、北极猎鹰树顶猎手（线膛后装击发枪狙击手，专杀军官）、鳟鱼怪物猎手（技工/捕鲸人，火箭弩猎飞鲸）。
- 战略学说：摒弃阵地战，纯散兵游击——把敌人拖入林海雪原，破坏指挥与补给使其溃散（「狮子也会被枭啄倒」）。
- 已知弱点：体量小、无法维持大军、缺野战重火力；猎手派军权过重、忠于保罗而非大技师；三派矛盾与邪教随扩张释放。

## 人格与行为逻辑（一致性锚点）
- 国家性格：冷酷、实用、殉道式重科学；以工场逻辑管理国家，拒不承认自己是「国家」。
- 核心矛盾 / 内部张力：议会派（还政议会、技术保守）vs 内阁派（大技师集权、激进不道德实验）vs 猎手派（军政府野心）；外加「不灭炉铸」邪教。
- 战略目标：大北进——寻找更易开采的浮晶矿/煤矿与有地热的新宜居地；防备西部势力。
- 红线：能量塔与大技师权威；为科学牺牲的传统不容质疑。
- 对待强者/弱者/玩家的风格：对土著强硬征服取劳力；对文明国家务实贸易；视技术机会高于道德，可为知识做交易也可做实验。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：三派内斗（扶植猎手派军政府或议会派起义）；提供禁忌技术/浮晶知识换取合作；「不灭炉铸」邪教可被点燃；少数民族（贝尔/枭/沃尔夫）作为反抗或雇佣资源；浮鲸/飞鲸猎场与浮晶矿作为利益诱饵。
