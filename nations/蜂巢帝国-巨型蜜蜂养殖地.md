# 巨型蜜蜂养殖地

> 蜂巢联盟成员子国。承载觉醒了「蜂巢意识」的巨型浮晶蜜蜂族群。

## 基础信息
- 政体 / 统治集团：格式塔 / 蜂巢意识（集体心智统治，由养蜂人侍奉真正的蜂王/蜂巢意识）。
- 首都 / 核心领土：蜂巢意识亲自指定的一座岛屿，因当地盛产海量「巨刃草」而迁居于此。
- 人口 / 经济基础：与蜂巢意识交流过的优秀养蜂人 + 一支舰队 + 「亿只」巨型蜜蜂；经济靠大量出口优质巨型蜜蜂蜂蜜，通常获利丰厚。
- 立国神话 / 历史定位：蜂巢共和国与蜂巢帝国内战期间，海神教与蜂群教剧烈反应致境内浮晶蜜蜂连同蜂巢巨大化，曾击退蜂巢共和国数次进攻；战后蜂群觉醒自我意识，新蜂后忧其威胁，蜂巢意识主动提出迁族，遂成此地。
- 主要种族：巨型浮晶蜜蜂（格式塔意识体）+ 人类养蜂人。

## 军事力量
- 常备军规模与构成：一支常规舰队（「巨型蜜蜂」舰队）+「亿只」巨型蜜蜂；蜂刺部门负责战时引导巨型蜜蜂上阵。
- 特色兵种 / 王牌：巨型浮晶蜜蜂蜂群——由常规舰队引导，海量蜂群近身撕碎敌人。
- 战略学说：常规舰队定位/引导，巨型蜜蜂蜂群作为主要打击力量进行群体歼灭。
- 已知弱点：技术与工业基础薄弱、缺乏远程/常规火力体系；战力高度依赖巨型蜜蜂这一单一生物资产。

## 人格与行为逻辑（一致性锚点）
- 国家性格：超然、统一意志的格式塔生命体，以延续蜂群与传统为念。
- 核心矛盾 / 内部张力：无内部政治生态（蜂巢意识统一）；外部张力源于其战力曾令蜂后忌惮而被迁离本土。
- 战略目标：守护巨型蜜蜂族群、维系蜂蜜产出与生存空间。
- 红线：巨型蜜蜂族群与其栖息地（巨刃草资源）安全不容侵犯。
- 对待强者/弱者/玩家的风格：以蜂群战力自卫，平时以蜂蜜贸易往来。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：巨型蜜蜂是蜂巢联盟最具威慑的生物兵器，谁能与蜂巢意识沟通/操纵谁就掌握筹码；威胁其栖息地或巨刃草资源可激怒之；优质蜂蜜是稀缺贸易品。
