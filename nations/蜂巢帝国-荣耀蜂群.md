# 荣耀蜂群

> 蜂巢联盟成员子国。由蜂巢内战时出逃的蜂巢帝国精锐舰队 + 数万民众，在旧帝国故地夺权立国。

## 基础信息
- 政体 / 统治集团：军人政权制（政变上位的舰队高层掌权）。
- 首都 / 核心领土：旧帝国曾经所在的大陆，一处原为旧帝国叛军政府的着陆区/城市。
- 人口 / 经济基础：撤离的数万蜂巢民众 + 当地叛军人口；初期主要靠掠夺周边叛军势力的资源与人口，自身生产力低下，战后转为出口劳动力维生。
- 立国神话 / 历史定位：蜂巢共和国与蜂巢帝国内战时，一支新式蜂巢舰队奉命携数万民众撤离（间接导致蜂巢帝国战败）；抵旧帝国故地后助当地叛军守城换取自由行动权，随后叛军高层政变、舰队实际掌权，立志复仇粉碎「蜂巢共和国」的进攻舰队。
- 主要种族：蜂巢帝国亡国遗民 + 当地（旧帝国叛军）居民。

## 军事力量
- 常备军规模与构成：陆军仅约2000人；一支规模庞大的精锐常规舰队 +「一支数量更为惊人的"舰队"」（疑似蜂群/特殊兵力）。掠夺部指挥「亡国」舰队，复仇部指挥「复仇」舰队。
- 特色兵种 / 王牌：传承蜂巢帝国的精锐舰队文化；全民皆兵（农夫亦受训）；无任何底线的反人类战法。
- 战略学说：精锐舰队 + 蜂群式数量优势；无禁忌、不择手段、不留退路的总体战。
- 已知弱点：陆军极少（2000）；自身生产力低下、依赖掠夺补给；远离本土孤悬旧帝国故地。

## 人格与行为逻辑（一致性锚点）
- 国家性格：背水一战、复仇至上、不择手段的亡国遗民军政权。
- 核心矛盾 / 内部张力：复仇派（主流，欲灭蜂巢共和国） vs 安居派（来都来了不如安居） vs 投降派（少数，主张投降蜂巢共和国）。
- 战略目标：早期一切战备皆为粉碎「蜂巢共和国」的进攻舰队，复仇雪耻。
- 红线：无战争底线、蜂群没有退路；任何动摇复仇大业者皆为敌。
- 对待强者/弱者/玩家的风格：对敌不择手段、掠夺周边；视复仇对象为不共戴天。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：极端复仇意识可被引导对准任意目标（曾针对蜂巢共和国）；安居派/投降派是内部裂痕；其掠夺习性令周边叛军势力天然敌对，可被联合围剿或反向收编。
