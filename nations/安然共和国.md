# 安然共和国

## 基础信息
- 政体 / 统治集团：民主共和制；议会为政治核心（每5年举行），12人领导层中选总统，军事事务由军事代表选出"军事总领"主导。
- 首都 / 核心领土：坐落于奥尔顿登西山脉东南部半岛；丘陵与低空浮空岛共存，连接地表与浮岛的建筑常见。
- 人口 / 经济基础：待补（人口数未明）；手工业发达，浮岛厂房林立；收入为工业税、贸易税、浮晶产品出售与旅游业（古文物纪念品），支出以购入矿物原料为主。
- 立国神话 / 历史定位：旧帝国中期与敌在达尔尼沃海大战留下无数残骸，帝国胜后于此建探索基地回收残骸，演变为探险/手工业城镇群；旧帝国解体后由原定居者建国，经多次扩张改革成今日共和国。
- 主要种族：待补（文本未明确，崇尚探索的多元社会）。

## 军事力量
- 常备军规模与构成：军部（战时准军政府）统辖；陆军为武装探险者与半脱产民兵（前装火枪、自制爆炸物）；防卫局守备浮晶岛十余门24磅前装炮与大量火箭礼花；海岸警卫队14艘小型桨帆巡逻艇（各装1门15磅前装炮）。
- 特色兵种 / 王牌：武装探险者民兵；实验性空军（研究浮晶稳定空中飞行）。
- 战略学说：国土保卫与自卫为主，依托浮岛地形与火炮要塞防御。
- 已知弱点：军备落后（前装时代）、海军仅桨帆小艇、空军尚无战力、依赖进口矿物原料。

## 人格与行为逻辑（一致性锚点）
- 国家性格：守成、探索导向、相对中立偏务实。
- 核心矛盾 / 内部张力：自立派（中立自卫）vs 联合派（对外经贸）vs 鹰派（军事扩张）三派博弈。
- 战略目标：发展国内产业、保有自卫力量、维系探索/手工业声誉与旅游经济。
- 红线：本土与浮空岛遭入侵、矿物原料供应线被切断、古遗迹/探索利益被外部掠夺。
- 对待强者/弱者/玩家的风格：对探险者与研究者友好优待；对外多走经贸友好路线，但鹰派可能推动武力获取境外利益。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：旧帝国海战遗迹与古残骸是探险/夺宝钩子；三派政治可被玩家影响（拉拢联合派经贸、煽动鹰派、安抚自立派）；浮晶产品与稳定飞行技术为合作筹码；激怒点为掠夺遗迹或切断矿物供应。
