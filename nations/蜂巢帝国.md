# 蜂巢帝国

## 基础信息
- 政体 / 统治集团：选举君主制；现实权力分散于蜂后派、议会派、激进派、改革派四大派系，议会派（贵族联合 + 新型资本家）势力最大但优势日衰。
- 首都 / 核心领土：首都兹拉莱多（兹拉莱多）；领土含大量附庸国与岛屿，分布于多个港口城市。
- 人口 / 经济基础：人口以本族为主，外族稀少且集中于港口城市；经济靠税收与海上贸易，主要出口食盐与（浮晶）蜂蜜，各派系各有贸易舰队，收入分配极不均。
- 立国神话 / 历史定位：蜂神乌塔布在远古混沌、神战致天昏地暗时，以泥土、人类希望与人心之光在夜空构筑发光蜂巢，赐恩人类；浮晶蜂群被视为蜂神后代。蜂巢帝国为旧帝国体系崩解后的继承性强权，后牵头组建「蜂巢联盟」。
- 主要种族：以本族（无显著外族）为主；少量外族定居于港口城市。

## 军事力量
- 常备军规模与构成：无统一陆军；由议会派贵族私军、蜂后派帝国海军/空军、改革派雇佣兵与自由民兵团、激进派「黄金蜂巢」地下武装等多支小型舰队拼成。海军陆战队为实际陆战主力。
- 特色兵种 / 王牌：海军陆战队接舷战与市区战专精（钩索跳帮快速突击）；蜂后之意特种舰队、蜂巢之主舰队、帝国边疆舰队、蜂群意识舰队。
- 战略学说：飞艇/海军主导，非对称外战经验丰富；登舰跳帮接舷战夺取地面与舰艇控制权。
- 已知弱点：力量碎片化、无中央军令；蜂后海空军被讥为「兹拉莱多海岸警卫队」、空军被贵族成建制收编；金色恐怖后内乱频发。

## 人格与行为逻辑（一致性锚点）
- 国家性格：表面统一的衰落帝国，实则四派暗斗、随时可能内战；对外以蜂群/联盟之名行扩张与镇压。
- 核心矛盾 / 内部张力：贵族议会派 vs 资本激进派 vs 复权蜂后派 vs 改革派；土地兼并、私军并立、蜂后兵权旁落。
- 战略目标：议会派维持优势、蜂后派夺回兵权、激进派废除贵族制、改革派接管议会；整体对外维系蜂巢联盟主导权。
- 红线：浮晶蜂蜜走私（蜂翼海关严查）；任何重演「破碎血晶」叛乱级别动乱的势力。
- 对待强者/弱者/玩家的风格：对附庸高压镇压、对盟友提供防御协助；对外来强权则贸易交流并戒备。

## 在玩家叙事中的作用
- 可被玩家利用/激怒/拉拢的钩子：四派内斗是天然杠杆——玩家可向激进派「黄金蜂巢」/改革派/蜂后派任一派系输送武器、资金或情报，挑动内战或扶植代理人；可借浮晶蜂蜜/食盐贸易渗透；可通过蜂语部门（兼间谍机构）展开外交或反间。
