# 伊帕坦努贾勒齐亚帝国

> 玩家国（新加入）。本档案记录其客观国力与开局既成事实；玩家意图由玩家/GM 掌握。

## 基础信息
- 政体 / 统治集团：绝对君主制。**国王**集立法/行政/司法/军事大权于一身，经"御前会议"（无固定成员，王室+高级贵族+重臣，决议以谕令/敕令发布）。下设外交/陆军/海军/王室内务/财政五大臣。
- 司法：高等法院（贵族控制、可制衡王权）、总督法院、教会法院、领主法院。
- 三等级社会：第一等级教士（约1%，大量土地、免税、控教育/慈善）；第二等级贵族（约1.5%，免税+高官特权，分传统贵族与赐官贵族）；第三等级（约97%，含资产阶级/农民/工人/城市贫民，承担几乎全部税收但政治权力极小）。
- 首都 / 核心领土：欧尔色内（第一帝国重建的旧帝国废城）；主体在西弗洛菲亚斯大陆，另有第四帝国夺取的"离岛区"。
- 经济基础：手工业末期（已出现较大规模手工场）+ 发达农业（大片平原）+ 浮岛空港采矿（煤/铁、静电浮尘分离装置）+ 风帆飞艇大陆级贸易。**国库亏空、通货膨胀、经济衰退**。
- 主要种族：**混血种 66%** + 人类 29% + 其余 5%。混血种为伊帕坦先民长期杂居融合而成，寿命略长（100–120 年），有长耳/假顶耳/尾/短翼等返祖性状；**魔法适性极差但魔法抗性极强（"魔法绝缘体"）**——源于西弗洛菲亚斯枯竭带。

- **技术等级（T0–T9 草拟）**：**T1**（复兴工艺阶段）——手工业末期、风帆飞艇而非动力飞艇、炼金未系统化、魔导仅有限的反魔法应用。特殊：因身处魔力枯竭带，走出了一条**反魔法（AM）+ 魔法绝缘种族**的独特技术路线。

## 军事力量
- 编制：集团军→军→师→团→营→连→排→班。
- 陆军：步兵/骑兵/炮兵多兵种。
- 陆军航空：风帆飞艇（重型运输 + 轻型对地轰炸），艇翼复杂、可操控性强，应对低空湍流。
- 海军：受"空中舰队"学说影响，已演变为以风帆飞艇为主、水面舰艇为辅；海军飞艇修长高速。第四帝国远征逐渐为军队/空中舰队配备专业反魔法术士。
- 已知弱点：财政崩溃、社会濒临革命、近30年对外战争连续受挫、整体技术等级偏低（T1）、魔法适性差。

## 开局前局势（推演准备｜开局 = 公历 1890 年）
档案本身停在一个极具张力的临界点（类法国大革命前夜）：第四帝国近30年对外战争皆败/僵持/议和 → 战争与贵族奢靡掏空国库 → 通胀加剧衰退 → 大众对皇室极度不满、进步思想加速传播 → **皇帝再次召开三级会议，试图向第三等级（97%人口）征税以救财政**。
> 这是开局的天然引爆点。待你定：开局年份与"瑟斯芬尼亚公历"的对应；三级会议的结果走向是否交由玩家行动决定；离岛区现状。
