# 铁血格瑞芬

> 玩家国。本档案记录其客观国力与开局既成事实；玩家意图由玩家/GM 掌握。
> 这是 `圣格瑞芬帝国` 经"德西娅·格瑞芬女皇维新"后的当前形态。**旧 docx = 前史，本档案 = 开局正典。**

## 基础信息
- 政体 / 统治集团：联邦制君主制。**帝皇 德西娅·格瑞芬**（格瑞芬二世之女）兼任**教皇**——皇权与神权合一。帝国内阁（门阀家族领袖+宰相，仍易瘫痪、无否决权）+ 维新后新设/强化的部门体系。
- 政治中枢：帝皇 → 帝国内政部（实际政务核心、制衡教廷）、帝国安全局（天眼，内外反间谍）、帝国大法院（审判贵族）、帝国财政部（维新后掌教廷税务）、帝国军务部（+ 天鹰纠察局反间谍）。秘密机构：督察团（帝皇私人秘密警察）、帝皇鬼手（刺杀机构，含科曼黑龙突击队、教廷圣裁者）。
- 首都 / 核心领土：阿宏雷拉斯（"狮鹫城"，旧帝国都城之一）；本土在克罗沃赛大陆西北、斯考福特山脉西部。
- 海外殖民地（弗里亚塔尤因大陆）：**十七自治领**（东南，最大，通商枢纽）、**克洛希雅第一直辖领**（中部跳板，直辖）、**最终教条区**（西部，"第二世界跳板计划"，备战黑锋、全自给自足、单向通讯、督察团监控）。
- 经济基础：27% 朝圣 + 73% 税收/商业；盛产巨木与火药、为周边代工飞艇/武器；**严重缺铁须进口**。维新后教廷垄断资产多收归国有，半数修道院/教廷学园改为国立学校。
- 主要种族：人类绝对主体；少数科曼黑蜥蜴人（水陆两栖、设自治区、部分为帝皇鬼手"科曼黑龙突击队"）。

## 军事力量
- **帝国军 = 铁血格瑞芬陆军 + 三空战旅 + 联合海军**：
  - 陆军梯队：**黑鹰师团**（精锐，约25000人/师，配巨型铁鹰动力装甲 + 速射火炮 + 后装枪，含武装神父）、**灰鹰师团**（中坚老兵，装备略逊、陨铁胸甲为主 + 火炮）、**银鹰师团**（新兵预备）、殖民地警备队、预备军。
  - 空战旅：第一支援空战旅（资深火力压制）、夏普洛斯-格瑞芬第二联合空战旅（含"圣殿"团异端自杀飞艇战术）、钢铁铸造者第三空降旅（步行机跳帮/空降、新型步行机驱逐舰）。
  - 联合海军：继承旧帝国风帆舰队，主力三级战列舰；殖民后规模扩大。
- 精锐/特殊：狮鹫团（帝皇约300人狮鹫骑兵卫队，维新后扩编兼军校教官）、新月骑士团（教廷狂热骑士+护教军+战斗修女，月光系战斗魔法）、帝皇鬼手（刺杀+科曼黑龙突击队）。
- 防御：继承旧帝国沿海炮台 + 内陆堡垒体系。
- 已知弱点：**缺铁严重制约火器/火炮量产**；内阁仍门阀化；神权狂热残余暗流（图谋证明教皇被逼让位以复辟教廷）；殖民地（尤其最终教条区）远悬海外、靠单向通讯与督察团维系、惧海上封锁；起义军骚扰。

## 开局前局势（推演准备｜开局 = 公历 1890 年）
据新档案回推，**开局前应已发生的因果链**：
1. **格瑞芬一世/二世之后，德西娅继位**：以"铁血手段"推动维新，改国名圣格瑞芬→铁血格瑞芬，政体改联邦制，统一克罗沃赛大陆西北部。
2. **解决皇权/神权死结**：迫使（前）教皇让位、女皇兼任教皇并让其子嗣世袭，神权实权收归帝皇；教廷资产收归国有、修道院/学园国立化。旧档案中"皇权 vs 神权内战在即"的引线，**以皇权完胜收束**。
3. **结盟与备战黑锋**：交结夏普洛斯（→瓦夏）为盟友后，对弗里亚塔尤因大陆上的黑锋帝国转为戒备，推行"第二世界跳板计划"，在弗里亚塔尤因建三殖民地（十七自治领/克洛希雅第一直辖领/最终教条区），最终教条区专为对黑锋开战做全自给自足的战争准备。
> 待你定：开局年份；维新/统一战争的烈度与时长；前教皇让位是和平交接还是流血；神权残余复辟暗流在开局当下的活跃度；最终教条区对黑锋的备战进度。

## 待澄清的一致性问题
- **同盟三角矛盾**：铁血视黑锋为备战对象，而 `瓦夏联盟` 称与"格瑞芬、黑锋"同时结盟——需你裁定这条三角关系。
- **地理命名**：旧档案称本土在"**菲尔佐甘大陆**西部"，新档案称"**克罗沃赛/克罗沃塞大陆**西部"，殖民地在"**弗里亚塔尤因大陆**"（≈瓦夏所在"弗里亚尤因"=洛德法尼亚？）。这些新地名与 `setting/geography.md` 的五大洲框架需要你对齐（是别称、改名，还是新测绘？）。
