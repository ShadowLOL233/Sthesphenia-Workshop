# 瓦夏联盟

> 玩家国。本档案记录其客观国力与开局既成事实；玩家意图由玩家/GM 掌握。
> 这是 `夏普洛斯` 经"米勒内战 + 安其拉社会主义革命"后的当前形态。**旧 docx = 前史，本档案 = 开局正典。**

## 基础信息
- 政体 / 统治集团：联邦制共和国（国家主义安其拉社会主义）。工人阶级依托公民武装领导的多党派"安其拉"政治联盟。
- 政治中枢：**安共会**（安其拉政治共同联盟总大会，最高立法/法理权力，5 年一次总代会）→ **安共中央**（常设监督）；**联政院**（最高执行）；**联战委**（军事统帅 + 最高政治决策 + 外交，司令≈总统，10 年任期）；**国家最高联合法院**（独立司法）。
- 执政党：**国家主义安其拉社会工党（安工党）**——保留市场经济为主体的法团主义引导经济、国有资本控制战略企业，长期目标无产阶级专政。
- 首都 / 核心领土：原夏普洛斯城（斯基普托纳海沿岸，温带森林，富铁矿）+ 扩张后的瓦兹拉克地区。
- 经济基础：市场经济（企业须经"法团"对接发展委）+ 国有龙头企业 + 计划工业化（5 年计划）+ 国家统配农业。重视外贸但不理想。
- 主要种族：人类为主（原住民 + 前帝国人 + 拉丁化夏普洛斯文化）。

## 军事力量
- 人民联合军（PU），下设四军种：
  - **国家革命军（NS，陆军）**：原瓦夏联合军，继承亚米乌斯义军传统，作风优良；战术以穿插包围运动战 + 短促突击为主；改革后大量装甲突击力量；基本单位师→团。
  - **中央联合军（CU）**：统帅部直属旅级特勤（约一个集团军），空中突击 + 机械化，战略决战决定性力量；下辖**国家总军特种部队（NGSF）**。
  - **联合防空军（UA）**：海空合并的新兴技术兵种（空军为主），战列巡洋舰为主力舰，全域三维作战。
  - **战略打击部队（SS）**：长程大威力机动炮兵，打击要塞/工业城市/决战兵器。
- 军政特务：联合军保安处（UG，反间谍/军事机密）、特别行动署（NSD，情报外勤）、武装公民革命战线（ACR，军宣+民兵训练）。
- 已知弱点：政权年轻、意识形态派系分裂（世界工党主张联俄/放弃法西斯同盟，与现行国家主义安全战略冲突）；联合战线极右恐袭持续；外贸薄弱。

## 开局前局势（推演准备｜开局 = 公历 1890 年）
旧档案明确埋下引线："新的风暴正在酝酿……"、激进派"将会获得千载难得的机会"。据新档案回推，**开局前应已发生的因果链**：
1. **米勒堕落 → 米勒内战**：卡西乌斯·米勒巩固个人权力、走向偏激，秩序部/情报部镇压旧起义军功臣，激进派与米勒派矛盾总爆发，演为全面内战（"米勒内战"）。
2. **第三党派崛起**：索迪纳诺斯·安其拉·杜米特库斯率领工人武装组建安工党（第三党），既反米勒寡头、也反亚米乌斯古典共和式"民主"，主张工人阶级专政。
3. **革命胜利 + 对外扩张**：安工党在内战中胜出，吸纳瓦兹拉克地区，建立瓦夏联盟；革命期间多次迎头痛击周边强敌，锻造出运动战传统的人民联合军。
4. **缔结同盟**：与格瑞芬（铁血格瑞芬）、黑锋结成军事同盟。
> 待你定：开局年份；米勒内战的起止与惨烈度；瓦兹拉克具体并入方式；联合战线恐袭与世界工党激进化在开局当下的活跃度。

## 待澄清的一致性问题
- **同盟矛盾**：本档案称"与格瑞芬、黑锋结军事同盟"，但 `铁血格瑞芬` 档案称其"最终教条区"是**进攻黑锋**的跳板、视黑锋为潜在大敌。瓦夏同时是格瑞芬与黑锋的盟友，而格瑞芬正备战黑锋——这条三角关系需要你裁定（瓦夏是中间斡旋方？还是设定冲突？）。
