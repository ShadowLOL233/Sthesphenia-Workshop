# _archive/ — 归档区（非活跃正典）

存放**已被取代/已退出活跃推演**的档案，使 `nations/`、`players/` 的活跃工作集保持精简（搜索、阅读、生成都更快）。归档文件仍保留以供**溯源**，但不计入活跃聚合表与开局正典。

## 子目录
- `前身存档/` — 已转变国家的"前身"档案（继任者在 `players/` 或 `nations/`）。
  - `圣格瑞芬帝国.md` → 继任：`players/铁血格瑞芬.md`
  - `夏普洛斯.md` → 继任：`players/瓦夏联盟.md`
- `工作日志/` — `工作日志.md` 滚动归档的旧开发日志条目（按现实月份分卷，如 `2026-06.md`）。

## 约定
- 归档时：在档案顶部保留"⚠️ 已被取代/已归档"横幅 + 指向继任者的链接；更新内部相对链接（`../` → `../../`）。
- 生成脚本（`scripts/gen_nations_table.py`）**不扫描** `_archive/`，故归档不影响活跃聚合表。
- 瀚朝、南洋都护府等归 GM 的"天下四分"体系管理，暂留 `nations/`，未归档。
