# 推演裁定规则 (Resolution Rules)

> 本文件由 GM 助手依据 `rules/文游推演机制 v3.0.1.docx` 与 `rules/GM使用指南.docx` 整理为可操作速查。源 docx 为权威，本文件是裁定时的快速参照。**这不是胜负系统，而是让历史显得"理应如此"的叙事参照框架。**

## 一、三层结构总览

| 层级 | 回答的问题 | 谁管 |
|---|---|---|
| **国家面板** | 你是谁、你擅长什么 | 国家属性（相对稳定） |
| **技术等级 (T0–T9)** | 你背负怎样的历史与结构压力 | 国家属性 + 结构性代价 |
| **稳速器系统** | 这个时代会如何回应你的行动 | **GM 专属，玩家不可操控** |

口诀：**技术层级决定你能做到什么，技术循环阶段决定时代是否允许你这样做，结构性乘数决定你为此付出的代价。**

## 二、国家面板（7 词条，0–10）

| 词条 | 含义 |
|---|---|
| 行政治理 | 能否把决定落实下去 |
| 经济基础 | 靠什么维持运转 |
| 军事能力 | 能否施加强制性力量 |
| 社会整合 | 内部是否稳定、能否承载压力与变革 |
| 科研潜力 | 能否产生新工具/新方法 |
| 对外影响力 | 能否影响他者 |
| 技术体系能力 | 能否把想法变成规模化现实 |

- **开局点数池 30 点**自由分配；单词条建议 **≥2**（避免结构空白）、**≤8**（避免开局极端化）。
- 目的是塑造"强项与短板"，不是全面最优。
- **不可频繁改**：只有重大事件后调整（成功的长期改革、技术代际跃迁、制度崩溃），且由 GM 在叙事中决定，玩家不能自行申请。
- 高技术等级国家开局须**压缩非核心词条点数**（体现资源高度集中、结构失衡）。

## 三、稳速器系统（GM 管理，世界级）

不代表国家能力，代表"这个时代允许什么发生"。开局通常设为**中等偏高张力**。

- **危机密度**：世界整体紧张度与冲突易发性。高 → 放大行动后果，小事变时代级事件。
- **社会适配度**（可降为单国/区域的次级判断）：制度—技术—社会的协调度。决定成功**是否付代价**。
  - 0–1 社会失序 / 2–3 低压 / **4–5 高压（瑟斯芬尼亚最常见也最危险）** / 6–7 崩溃临界·反噬边缘。
- **技术循环阶段**（核心，世界级，决定时代对创新的容忍度）：
  - **消化期 (0–1)**：技术被吸收标准化，可靠但不激动人心，适合长期建设。
  - **推进期 (2–3)**：新技术频出且被迅速军事化，机会与隐患并存，军备竞赛高发。
  - **过热期 (4–5)**：高负荷运转，事故/失控/伦理争议集中爆发，资源被迫转向维护。
  - **停滞与反噬期 (6–7)**：新技术骤减，结构性事故常态化，部分技术被封存遗忘，孕育下一轮突破。

## 四、结构性乘数（长期代价，不显性计算）

- 只用于**关键行动**（技术突破、超级工程、战略级战争、制度性改革），不用于日常。
- 不把成功变失败，而是**改变成功的成本**——以额外条件/代价/后果写入结果（如过热期推突破 → 成功但伴随事故/资源虹吸/社会裂痕）。
- 必须可被叙事理解（即使不公开数字，也要说明为何此时此类行动难/危险）。
- 高技术国家后续提升/改革时更易遭遇失败或反噬。

## 五、回合与行动

- **1 回合 = 1 游戏年**，三阶段：**行动提交期 → 裁定期 → 年纪事发布**（各阶段时长每回合公告）。
- 每国每回合**最多 3 条主要行动**。联名行动占用每个参与国各自 1 个名额（参与一条联名 → 本回合只剩 2 条独立行动）。
- "维持现状"不算行动，是默认状态；不行动的领域按面板能力自然运转。

## 六、行动判定公式

1. **选词条**：从 7 词条中选最相关的一个。跨多领域时**通常取较弱词条**（薄弱环节决定可行性）。玩家可建议，GM 最终定。
2. **定难度**：

   | 档位 | 目标值 | 情境 |
   |---|---|---|
   | 简单 | 6 | 符合国力、时代配合、无重大阻力 |
   | 中等 | 8 | 标准难度，多数行动落此档 |
   | 困难 | 10 | 逆时代、对手强大或短板明显 |
   | 极难 | 12 | 几乎超出国家能力边界 |
   | 几乎不可能 | 14 | 违背时代逻辑或试图越级跨越结构性代价 |

3. **投骰**：GM 掷 **2d6 + 相关词条**，与难度比较。`≥ 难度 → 成功`，`< 难度 → 失败`。
4. **特殊**：裸 12（双 6）= **大成功**（达成目标 + 额外正面后果）；裸 2（双 1）= **大失败**（失败 + 额外负面后果）。
5. **叙事化呈现**：不直接报"你掷了 9，失败"，而把结果融入年纪事——展示事件如何展开、为何如此、留下什么后果。

> 结果不止成功/失败二态，更可能是：有代价的成功、失败但留下长期影响、引发新危机或连锁反应。

## 七、行动提交模板

```
========================
【国家名】回合 N 行动书
公历 XXXX 年
========================
【行动 1】
标题：一句话概括
动机：为什么要做
依托条件：依靠什么（面板哪些词条？资源/关系/历史背景？）
期望方向：希望达成什么（只需方向性目标，不必承诺数字）
【行动 2】…
【行动 3】…
【备注】（可选）私下协商成果、对前回合年纪事的反应等
========================
```

## 八、信息分级

- **公开层**：所有玩家可知（各国基本国情、年纪事、公开外交关系）。
- **国家层**：本国可知，他国需情报行动获取（面板具体数值、内政细节、秘密外交、隐秘行动）。
- **世界层**：GM 专属（月亮、大灾变、行星环、远古文明的更深真相）。角色初始不知，只能通过扮演逐步浮现。

## 九、GM 立场

- GM 是**世界叙事的管理者与协调者，而非裁判或对立方**。目标是维护世界逻辑、时代节奏与叙事连续性，而非保证任何单一国家成功。
- 默认 GM 给出推演结果与推理依据，**GM（你）拥有最终否决/调整权**。
- 失败不等于无意义；成功常伴隐性成本。衡量行动看的是对世界结构与故事走向的影响，而非是否达成初始目标。
