# 核心设定 (Setting)

> 本文件由 GM 助手依据 `_extracted/` 中的设定源文档（世界观概述第一章、故事背景、技术层级、浮晶设定、种族基础设定等）整理。源 docx 仍保留在 `setting/`，本文件是供推演快速引用的索引摘要。

## 世界基本盘

- **世界名**：瑟斯芬尼亚 (Tharsphennia / Suspendiumnia)，得名于核心资源**浮晶 (Suspendium)**。
- **行星**：巨型大陆世界，地表面积约为地球三倍；**三颗月亮 + 行星环**；重力与磁场接近地球，生态多与地球相似并有独特本地物种。
- **当前纪元**：**开局 = 公历 1890 年**（回合 1）。世界观文献亦称此为"大灾变后约 147 年"的"双重重建"时代；公历与大灾变纪年的精确换算待 GM 确认。
- **基调**：旧秩序崩解、新秩序未立的动荡上升期——"什么都可能发生，但没人保证飞艇不会下一秒坠落"。王国让位共和国、贵族让位企业、教会让位学会。底层悬着一个共同恐惧：**大灾变会不会再来**。

## 技术时代：双重重建

魔法与工业并行且交织，两者都被各自学者视为可科学研究的对象。

- **魔法的重建**：旧帝国失传的法术/仪式/理论，靠废墟残片一点点重新拼凑（进度极慢，高等精灵学者耗几十年才读懂数页手册）。
- **工业的初建**：蒸汽机、铁路、炼钢炉、浮晶飞艇——全是大灾变后从头发展的体系，与旧帝国的原理无关。
- 三条技术轴：**工业、炼金、魔导**（评级见技术层级 T0–T9，下文「技术等级」一节）。

## 浮晶 (Suspendium) — 文明支柱

- 淡蓝/青绿/银白色晶体，存在于空中浮岛、地底岩层及大气微粒中。**类金属导电性**。
- **通电充能 → 以晶体为核心生成反重力立场**，削减立场内物体的有效重量，使大型载具/建筑得以低空航行或半悬浮。电流中断则立场逐步消散。
- 立场强度取决于**晶体质量、几何完整度、内部层纹一致性**；特定几何排列的多晶可形成**稳定共振**，综合升力远超单晶。
- **关键约束**：浮晶**不能**作魔力石用；其**粉尘会干扰/破坏环境魔力流动**并造成长期污染（充能时粉尘被立场抑制，立场消散后扩散）。
- **战略含义**：浮晶文明本质是**加工文明**——国力强弱不只看矿脉，更看晶体学、切割、精密制造与维护能力。飞艇升空发生于约**23 年前**，自此天空成为可经营、争夺、统治的新边疆。

## 超自然 / 魔法规则

- 魔力源自世界本身；所有生物体内都含魔力，魔力可跨种族/物种形成"枢纽"。
- **种族魔力适性差异巨大**：高等精灵适性极高（魔法渗透社会方方面面），劣种半兽人几乎无感知。非类人生物的智慧与其接入魔力枢纽的程度相关——许多魔兽/野兽被驯为牲畜、坐骑、伙伴。
- 大灾变曾导致**魔力枯竭 + 魔力潮汐**（施法者受不可逆伤害），约一百年后、距今约半世纪才平息。

## 种族（概览）

精灵、四足人、矮人、鱼人、羽翼人、半兽人、蜥蜴人、巨人、血族、龙族及亚种等。详见 `setting/瑟斯芬尼亚种族基础设定.docx`（约 3.2 万字，未在本摘要展开）。

## 技术等级 T0–T9（评级标尺）

三轴（工业 / 炼金 / 魔导）共用的宏观阶段。开局国家通常落在 **T1–T3**。

| 级 | 名称 | 一句话 |
|---|---|---|
| T0 | 知识复兴与秩序重建 | 活下来优先；知识碎片化、依赖个人天赋，遗产物件只会用不懂原理 |
| T1 | 复兴工艺 | 蒸汽与可控热能出现；火枪连队、蒸汽炮；炼金/魔法仍是附属 |
| T2 | 动力工业起始 | 可控动力稳定、机械可重复；炼金标准化量产；中等常备军 |
| T3 | 体系工业成型 | **三轴首次同时工程化**；法阵标准化、浮晶工程化突破；空域与战争形态被重塑 |
| T4 | 高复杂工程 | 跨区域基建常态；炼金介入生命/灵魂/物质重构，构造体入伍 |
| T5 | 稳定体系部署 | 技术固化为可靠但脆弱；创新被迫转入隔离/军工/保密机构 |
| T6 | 战略工程 | 工程直接重塑地理/空域；巨型构造体常规化；失败 = 体系级崩溃 |
| T7 | 三轴全面融合 | 工业/炼金/魔导不可区分；"不可局部失败"，多毁于自身协调极限 |
| T8 | 遗产技术再现 | 稳定使用部分灾变前遗产；高度发达却极不自由，被自身无法再现的成果束缚 |
| T9 | 大灾变源头文明 | 仅描述导致大灾变的原初/守护者级存在；**不对任何玩家或国家开放** |

> 技术等级不等于强弱，而是**结构性代价**：越高级，制度越复杂、社会张力越大、改革越易反噬（详见 `rules/resolution-rules.md`）。

## 大灾变（背景真相未知）

旧帝国（统称，可能非同一文明）在约 147 年前一夜覆灭，成因众说纷纭（魔法战争 / 神罚 / 远古存在苏醒 / 旧文明科技灾难），无一可证实或证伪。三大确证影响：**魔力枯竭与潮汐**、**生态剧变与"死地"**、**知识与文化断层**。

> **信息分级**：关于月亮、行星环、大灾变、远古文明的更深真相属于**世界层（GM 专属）**，不通过任何文档预先告知玩家，只能在扮演中逐步浮现。

## 玩家与 NPC 的关系基调

- 玩家是世界观主导者；NPC 用于辅助和推进玩家叙事，需有自洽动机但不抢主线。
- 这不是胜负游戏，而是集体历史书写：国家会兴盛也会衰落，计划会成功也会失败。
