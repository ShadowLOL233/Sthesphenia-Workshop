# 历史事件时间线

> 按时间排序的已发生事件。每条记录因果链，便于一致性校验。**只记已落库的"既成事实"**；推演中的假设/分支不进此文件。

## 记录格式

```
### [推演时间] 事件标题
- **触发**：（玩家行动 / NPC 行动 / 随机事件）
- **经过**：
- **直接后果**：
- **次级反应**：谁被激怒/受益，产生了哪些新矛盾或 casus belli
- **影响文件**：更新了 nations-table / relations / 哪些档案
- **相关冲突**：conflicts/xxx.md（如有）
```

## 时间线

> 最新事件置顶或置底由 GM 习惯决定；默认按时间正序，新事件追加到末尾。
