# 外交关系矩阵

> 记录国与国之间的态度与趋势。每次重大事件后更新。玩家国通常放在最前列，因为它们是叙事焦点。

## 态度标尺

| 符号 | 含义 |
|---|---|
| ⚔ | 敌对 / 交战 |
| ⚠ | 警惕 / 紧张 |
| – | 中立 / 无往来 |
| 🤝 | 友好 |
| 🛡 | 同盟 / 保护关系 |

趋势箭头：`↑` 改善中　`↓` 恶化中　`→` 稳定
