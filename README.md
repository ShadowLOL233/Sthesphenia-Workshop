# 瑟斯芬尼亚 (Tharsphennia) 推演工作仓库

GM 全知视角的世界推演工作区。**玩家是世界观的主导者，NPC 国家存在的目的是辅助并推进玩家驱动的世界发展。**

## 仓库结构

```
tharsphennia/
  README.md            本文件：索引 + 当前世界状态快照
  setting/
    setting.md         核心设定（技术时代、地理尺度、超自然规则等）— 待你上传
    geography.md       地图分段、地形高低、生物群落的索引与说明
  rules/
    resolution-rules.md 推演裁定规则（军事/外交/经济）— 待你上传
  players/
    _template.md       玩家国档案模板
    <玩家国>.md         每个玩家国一份档案（主角阵营）
  nations/
    _template.md       NPC 国家档案模板
    <国名>.md           每个 NPC 国家一份档案
  nations-table.md     所有国家横向对比主表
  relations.md         外交关系矩阵
  timeline.md          按时间排序的已发生事件 + 因果链
  conflicts/
    _template.md       军事冲突推演模板
    <战役名>.md         单场冲突的兵力、过程、结果、政治余波
  maps/                地图图片存放区（大陆分段 / 地形 / 生物群落）
```

## 工作约定

- 任何新事件落库前先做一致性校验，发现与既有事实矛盾先报 GM，不硬写。
- NPC 行为遵循其档案中的人格/学说/目标，保持跨事件一致。
- 关系与世界状态变化后，同步更新 `relations.md`、`timeline.md` 和本文件的快照节。
