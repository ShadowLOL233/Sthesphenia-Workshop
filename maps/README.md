# 地图图片存放区

把你的地图图片放这里，建议命名规则便于索引（对照表维护在 `../setting/geography.md`）：

```
政治图：   political_<大陆/分段名>.png      例 political_north.png
地形高低图：terrain_<大陆/分段名>.png        例 terrain_north.png
生物群落图：biome_<大陆/分段名>.png          例 biome_north.png   （未完工标 _wip）
```

我能读取这些图片来辅助推演，但请在 `geography.md` 里确认地名与归属，避免我读图时误判边界。

> 注：若图片体积很大、数量很多，跨设备同步时可考虑 Git LFS；需要时告诉我，我帮你配置。
